Event management calendar

Overview
This project is a simple Python program designed to manage events in an airport. It allows the user to add and view airplanes landing or departing, add passengers in the plane and give seat numbers, book tickets for flights.

Features
-Add a new flight and passenger
-View all flights and passengers
-Book tickets
-View those bookings
-Update the status of the airplanes
-Stores data inside a list (In-memory database)
-The prerequisite data-structure for some important data will not give error when the input data-structure is wrong.

How to Run
1.Open terminal or command prompt.
2.Navigate to the folder where airport.py is saved.
3.Run the program using:python
